# Galería Interactiva con JavaScript

Este proyecto es una galería de imágenes interactiva desarrollada con HTML, CSS y JavaScript.

## Funcionalidades
- Agregar imágenes mediante URL
- Seleccionar una imagen con clic
- Eliminar la imagen seleccionada
- Diseño responsive usando CSS Grid

## Tecnologías
- HTML5
- CSS3
- JavaScript

## Publicación
El proyecto está publicado en GitHub Pages.
